## 🚀 Start in Just 3 Steps

### Step 1: Start Backend (Terminal 1)
```bash
cd backend
npm run dev
```
✅ Will show: "Server is running on http://localhost:5000"

### Step 2: Seed Database (Terminal 2 - Optional)
```bash
cd backend
node seed.js
```
✅ Will add 6 sample students

### Step 3: Start Frontend (Terminal 3)
```bash
cd frontend
npm run dev
```
✅ Will show: "Local: http://localhost:5173"

## ❓ Quick FAQ

**Q: What if I don't have MongoDB?**
A: MongoDB Atlas (cloud) is included in `.env` - just create a free account at mongodb.com

**Q: Can I change the admin password?**
A: Yes, edit `.env` and run `/api/auth/init-admin` again

**Q: How do I add more students?**
A: Click the "Add Student" feature (to be implemented) or use the API directly

**Q: Is it production-ready?**
A: Yes! See [DEPLOYMENT.md](./DEPLOYMENT.md) for cloud deployment
